A block-level element starts on a new line, and browsers automatically add some space (a margin) before and after the element.

A block-level element takes up the full available width (stretches left to right).

Here are common block-level elements in HTML:

<address> <article> <aside> <blockquote> <canvas> <dd> <div> <dl> <dt> <fieldset>
<figcaption> <figure> <footer> <form> <h1>-<h6> <header> <hr> <li> <main>
<nav> <noscript> <ol> <p> <pre> <section> <table> <tfoot> <ul> <video>
